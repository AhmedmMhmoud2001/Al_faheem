# Al Faheem Admin

React (Vite) + Tailwind v4. Port **5174** (see `vite.config.js`).

## Setup

```bash
cd al-faheem-admin
copy .env.example .env
npm install
npm run dev
```

Set `VITE_API_URL=http://localhost:4000/api/v1` and ensure the API CORS includes `http://localhost:5174`.

Login with seeded admin: `admin@al-faheem.local` / `Admin123456`.
